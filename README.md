# Momentum Mayhem

Momentum Mayhem is a momentum based minecraft minigame where "everything is changing". Players are thrown into an arena and must fight to be the last one standing while the game unleashes random disasters that alter gravity, speed, and even the floor beneath you.

It supports any amount of players. More players = more chaos

To play, download this [mod](https://github.com/hasware1/MomentumMayhem/releases/tag/minecraft_minigame) and the [world](https://github.com/hasware1/MomentumMayhem/tree/main/Fractal)

Upon joining players will be placed in a lobby, press the start button to start the game. Last one standing wins ;D
